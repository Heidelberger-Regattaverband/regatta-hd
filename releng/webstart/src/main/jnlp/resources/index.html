Install Heidelberger Regatta software

  1. Download and install OpenWebStart (x64 version).
  2. Download Java policy file and save it in user's home directory (C:\users\<user>\.java.policy) with file name .java.policy.
  3. Download Heidelberger Regatta software and open it with OpenWebStart to install.

Install Aquarius MMX software

  1. Download and extract Aquarius ZIP archive
  2. Alternative download of Aquarius with Infoportal + Logos
  3. Documentation for Aquarius

Configure Aquarius MMX for Heidelberg Regatta

  1. Rename existing report <aquarius_install_dir>\Reporting\XSL\FOP\R_RE.xsl to R_RE.xsl.bak as backup.
  2. Then download modified Aquarius report (R_RE.xsl) for Einteilung am Sonntag and save it at location <aquarius_install_dir>\Reporting\XSL\FOP\R_RE.xsl.
  3. Configure keyboard shortcuts for start and stop in Aquarius's config file: C:\Users\<user>\AppData\Roaming\Aquarius\aquarius.ini: 
    4. [Timing]
ShortcutStart=+ (ZEHNERTASTATUR)
ShortcutFinish=Leertaste

Infoportal Heidelberger Ruder-Regatta

  1. Link to Infoportal
  2. Embed Infoportal into HTML: 
    <iframe src="https://info.regatta-hd.de/infoportal/" title="Infoportal" height="500" width="100%" />

Install Ampelkoffer software
