Heidelberger Regatta Software

  1. OpenWebStart herunterladen und installieren.
  2. Java Konfiguration herunterladen und im Benutzer Verzeichnis (C:\users\<Benutzername>\.java.policy) mit dem Dateiname .java.policy speichern.
  3. Heidelberger Regatta Software herunterladen und durch öffnen mit OpenWebStart starten.
  4. Infoportal zur Heidelberger Ruderregatta
