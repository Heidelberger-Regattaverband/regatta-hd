Install Heidelberger Regatta software

  1. Download and install OpenWebStart (x64 version).
  2. Download Java policy file and save it in user's home directory (C:\users\<user>\.java.policy) with file name .java.policy.
  3. Download Heidelberger Regatta software and open it with OpenWebStart to install.

Install Aquarius MMX software

  1. Download and extract ZIP archive
  2. Dokumentation
  3. Download modified Aquarius report (R_RE.xsl) for Einteilung and replace existing report at this location: <Aquarius-Installation>\Reporting\XSL\FOP\R_RE.xsl.
    Rename existing report to R_RE.xsl.bak
  4. Configure keyboard setting for start and stop in Aquarius's config file: C:\Users\<user>\AppData\Roaming\Aquarius\aquarius.ini: 
    5. [Timing]
ShortcutStart=+ (ZEHNERTASTATUR)
ShortcutFinish=Leertaste
    Optional steps to enable encrypted connections to MS-SQL server
  5. Install Microsoft Visual C++ Redistributable (x64 + x86) as prerequisite of the OLE DB-driver
  6. Install OLE DB-driver to support encrypted connections to MS-SQL server

Additional notebook configurations

  1. Configure hosts file: C:\Windows\System32\drivers\etc\hosts:
    2. 217.160.56.230	mssql-aquarius		# 8e835d.online-server.cloud
82.165.106.228	mssql-aquarius2		# b3eb6e9.online-server.cloud
    The hosts file requires administrator permission to be modified. Follow this guide to edit it.
